# Customer Churn Prediction Analysis

## Project Overview
This project analyzes customer churn behavior using Excel and PostgreSQL.

The objective is to identify customers who are likely to leave the service and understand the factors affecting churn.

## Tools Used
- Microsoft Excel
- PostgreSQL
- SQL Queries
- Pivot Tables
- Pivot Charts

## Dataset Features
- Customer ID
- Customer Name
- Age
- Gender
- City
- Subscription Type
- Monthly Charges
- Tenure Months
- Complaints
- Support Tickets
- Churn Status
- Customer Category
- Churn Risk

## Key KPIs
- Total Customers
- Active Customers
- Churn Customers
- Churn Rate %

## Dashboard Insights
### Customer Churn Distribution
Shows percentage of churned and active customers.

### Churn by Subscription Type
Analyzes churn across Basic, Premium and Standard plans.

### Customers by City
Identifies cities with highest churn.

### Loyal vs New Customers
Compares customer categories.

### Complaints vs Churn Status
Measures relationship between complaints and churn.

## SQL Analysis Performed
- Total Customers
- Total Churn Customers
- Active Customers
- Churn Rate %
- Churn by Subscription Type
- Churn by City

## Business Benefits
- Reduce customer loss
- Improve customer retention
- Identify high-risk customers
- Support business decision making
